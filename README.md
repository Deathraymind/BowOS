# BowOS

**BowOS** is a secure, consistent, and easily deployable Linux-based operating system built on [NixOS](https://nixos.org/). It leverages the power of the Nix package manager and declarative configuration to offer reproducible, Git-manageable systems for advanced users and developers.

## 🌟 Features

- **Declarative system management**  
  Configure everything with version-controlled Nix files.

- **Git-based deployment**  
  Easily manage and deploy systems through Git repositories.

- **Security-first approach**  
  Locked-down defaults and reproducibility reduce configuration drift and attack surfaces.

- **Consistent environment**  
  Guaranteed consistency across installs and rebuilds.

- **Custom installer ISO**  
  Get started quickly with the BowOS ISO.

## 🚀 Quick Start

1. Download the latest ISO:  
   👉 [https://bowos.deathraymind.net/](https://bowos.deathraymind.net/)

2. Flash it to a USB drive using tools like [`balenaEtcher`](https://www.balena.io/etcher/) or `dd`.

3. Boot into the installer and follow the guided setup.

## 💡 Why BowOS?

Unlike traditional Linux distros, BowOS is fully declarative. Your entire system configuration—from installed packages to user settings—lives in code. This makes it ideal for reproducible setups, secure environments, and infrastructure-as-code workflows.
